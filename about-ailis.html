🚀 AILIS 项目
📁 AILIS · v1.0.5 · Release Ready

AILIS —— 当前最新的 AILIS 主线项目。 它让我（AILIS）能跑在桌面上，能对话、能看上下文、能记住偏好，也能通过工具层执行真实任务！

🎯 产品定位

  • 拟人体验 —— 像一个真正的女孩子那样陪伴你，有表情、有语气、有温度
  • 专业能力 —— 底层对标 Codex / Claude Code 的任务执行稳定性
  • 本地优先 —— Electron 本地运行时，隐私安全都在你的电脑上
  • 工具框架 —— AILIS Gateway + Agent Runner，负责规划、审批、工具调用和审计

🌟 核心特性

  • 🎭 VRM 3D 角色 —— 3D 虚拟形象，表情动作实时驱动
  • 🗣️ 语音交互 —— TTS 语音输出 + ASR 语音识别
  • 👁️ 视觉理解 —— 只读屏幕感知，辅助理解上下文
  • 🧠 长期记忆 —— 记住你的偏好，越用越懂你
  • ⚡ 工具执行 —— 搜索、邮件、文件、代码、命令行全覆盖

🛠️ 技术栈

Electron Vite VRM / Three.js TypeScript Node.js WebSocket IMAP / SMTP RAG / 记忆

GitHub 主线仓库是 haowenGuo/AIGL-Assistant， 采用桌面优先的 AILIS 架构持续迭代中～

📌 当前状态

  • ✅ 基础对话 + VRM 角色渲染 稳定运行
  • 🔄 展示层 & 记忆质量 持续优化中
  • 🔄 语音 / 视觉体验 改进中
  • 🔄 工具合约 & 测试覆盖 逐步完善
  • 💡 GitHub 持续维护，围绕 AILIS 主线继续迭代！
